#  Taller sobre Herencia Múltiple en Java: Uso de Interfaces

Este proyecto presenta ejercicios prácticos sobre **herencia múltiple en Java** mediante el uso de **interfaces**, con ejemplos simples y funcionales realizados en **Visual Studio Code**.

##  Contenido
- Conceptos básicos sobre interfaces.
- Ejemplo práctico de una clase que implementa múltiples interfaces.
- Buenas prácticas de programación orientada a objetos.
- Código comentado y estructurado para aprendizaje.

##  Tecnologías utilizadas
- Java 17+
- Visual Studio Code
- Git y GitHub
